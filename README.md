
BDL CONSTRUCTORA
¡Bienvenido a BDL CONSTRUCTORA! Esta aplicación ha sido desarrollada por Ismael Liquez, estudiante de Ingeniería, como parte de un proyecto encargado. La aplicación está diseñada para gesitionar procesos administrativos.

## Características principales**
- **Desarrollada en C#**: La aplicación está construida utilizando el lenguaje de programación C#.
- **Base de datos MySQL**: Utiliza una base de datos MySQL alojada en un servidor en la nube para almacenar y gestionar datos.

- 
## **Requisitos del sistema**
- **Sistema operativo**: Windows 10 o superior.
- **.NET Framework**: Asegúrate de tener instalado .NET Framework 4.7.2 o superior.
- **Conexión a Internet**: La aplicación requiere conexión a Internet para interactuar con la base de datos en la nube.

---

## **Instalación**
1. - Ejecuta el archivo `setup.exe`.
   - Sigue las instrucciones del instalador.
   - Al finalizar, se creará un acceso directo en el escritorio para iniciar la aplicación.

2. **Desde un archivo descargado**:
   - Descarga el instalador desde [enlace de descarga, si lo tienes].
   - Ejecuta el archivo `.exe` y sigue las instrucciones del instalador.

## **Tecnologías utilizadas**
- **Lenguaje de programación**: C#.
- **Base de datos**: MySQL.
- **Servidor en la nube**: Aiven.IO.
